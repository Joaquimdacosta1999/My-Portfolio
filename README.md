# Joaquim Costa Portfolio

🚀 A professional portfolio website showcasing my skills, projects, and career journey.

## 🔹 Features
- Responsive design (HTML, CSS, JS)
- Smooth scrolling & animated typing effect
- Scroll-to-top button
- Modular sections for easy updates
- Links to GitHub projects & live demos

## 📂 Structure
- `index.html` → Landing page
- `sections/` → About, Education, Experience, Projects, Skills, Contact
- `assets/` → Images, icons
